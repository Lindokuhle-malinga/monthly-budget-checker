# Monthly Budget Checker

## Overview
The Monthly Budget Checker is a Python application designed to analyze monthly income against categorized expenses and evaluate financial stability.

This project applies structured financial reasoning using Python fundamentals.

## Features
- Accepts monthly income input
- Records categorized expenses (food, rent, transport, other)
- Calculates total expenses
- Determines remaining balance
- Evaluates financial condition using conditional logic
- Simulates multiple months for financial tracking
- Flags potential debt growth risks

## Technologies Used
- Python 3

## Financial Logic Applied
Total Expenses = Food + Rent + Transport + Other  
Remaining Balance = Monthly Income − Total Expenses

The program evaluates:
- Surplus (good control)
- Balanced budget
- Budget deficit
- Debt warning signals

## Purpose
This project demonstrates the ability to combine financial analysis concepts with programming logic to simulate real-world personal finance assessment.
